main About Back to Projects Blog Contact Resume

Problem Statement

Count the number of objects in the image.

Approach

  1. Used OpenCV libraries involving thresholding, edge detection, contour detection to count the objects.
  2. Used RCNN and SSD models trained on COCO dataset using Tensorflow Object Detection API

Results

  1. OpenCV performed great when there were non overlapping images.
  2. RCNN models outperformed OpenCV results and SSD trained networks on images with objects pretrained using COCO dataset.
